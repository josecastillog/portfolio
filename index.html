MARCO ARcade

MARCO ARcade is a native iOS Augmented Reality app designed and developed for the MARCO museum. The app features an interactive videogame aimed at engaging and educating children while they explore the museum. To ensure a seamless user experience, a physical space detection AR system was implemented to allow accurate user location tracking.

BandMate

BandMate is an iOS app that helps musicians find compatible bandmates. It features a chat client built with UIKit, MongoDB, and WebSockets. The app's music taste-based matching algorithm, powered by the Spotify API, enables musicians to connect with those who share similar tastes.

Historify

Historify is a native iOS app that allows users to view their personalized Spotify statistics. It was developed using SwiftUI, and the Firebase database. OAuth2.0 was used for secure user authentication with the Spotify API. This app provides an enjoyable and personalized experience for Spotify users to view their streaming history, within a secure and user-friendly interface.
